# QRT-datachallenge2025
My solution for the QRT 2025 Data Challenge: Predicting Overall Survival for Acute Myeloid Leukemia patients. Includes advanced feature engineering on clinical/molecular data and optimized ensemble survival models (Gradient Boosting, ExtraSurvivalTrees, RandomSurvivalForest) to maximize the C-Index
